# AboVagt.dk

Landing page for AboVagt – hjælper danskere med at finde og opsige glemte abonnementer.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deploy: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
